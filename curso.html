Temas del curso

Introducción a la Programación Web

Conceptos básicos sobre el desarrollo web, tipos de páginas y funcionamiento de internet.

HTML5

Aprendimos la estructura de una página web, uso de etiquetas como header, nav, section, article y footer.

Estructura de HTML

Organización correcta del código HTML, uso de etiquetas semánticas y buenas prácticas.

Enlaces en HTML

Uso de enlaces relativos y absolutos para conectar páginas web y navegar entre ellas.

Introducción a CSS

Aplicación de estilos a páginas web, uso de colores, fuentes y diseño visual.

Uso de cajas en CSS

Uso del modelo de cajas (box model), márgenes, padding, bordes y distribución de elementos.
